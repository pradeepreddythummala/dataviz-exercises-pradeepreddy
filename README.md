# FIFA World Cup Match Analysis Dashboard

## Project Overview

This project analyzes FIFA World Cup Men's Match data from 1970 to 2022 using Python, Pandas, Plotly, and Streamlit. The project includes Exploratory Data Analysis (EDA) and an interactive dashboard to visualize match statistics and team performance.

---

## Objectives

- Perform Exploratory Data Analysis (EDA)
- Analyze FIFA World Cup match statistics
- Build an interactive Streamlit dashboard
- Visualize key insights using Plotly

---

## Dataset

**Dataset Name:**
FIFA World Cup Men's Match Dataset (1970–2022)

The dataset contains information such as:

- Tournament Name
- Team Name
- Opponent Name
- Goals Scored
- Goals Against
- Possession
- Shots
- Shots on Target
- Corners
- Yellow Cards
- Stadium Name
- Match Result

## How to Run the Project

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run the Streamlit Dashboard

```bash
streamlit run dashboard.py
```
